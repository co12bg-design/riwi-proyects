#Inventory Pogram

##Description: This program is designed to validate the necessary information required to register an inventory. It is a tool that ensures product details are accurate before storing them. The program checks information such as product name, price, and quantity. It calculates the total cost and displays the results in a structured list along with the validated inputs.

##How it works:
1. The program requests the user to enter:
-product name
-product price
-product quantity
2. Each input is validated:
-Name must invlude only letters
-price must be greater than 0
-the quantity must be a positive integer 
3. If the user enters invalid data, the program display an error message and ask for the information again. 
4. Once all the inputs are valid, the program calculates:
The total cost: multipling the price and the quantity of the product.
5. Finally the program prints the infromation in a clear format.

>The program is currently running and validating inputs continuously to ensure accurate inventory registration.
